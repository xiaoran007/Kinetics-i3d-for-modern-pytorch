## Conversion and Parity

This release checkpoint is converted to canonical state_dict format and validated with strict parity checks:

- strict `allclose(atol=0, rtol=0)` on fixed random input (`clip_logits`)
- strict `allclose(atol=0, rtol=0)` on sample clip (`clip_logits`)
- sample top-k ordering consistency between source and canonical models
- canonical reload parity consistency

See attached report files for exact values and hashes.
